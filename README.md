## AnyaEngine
- ```AnyaEngine``` 是一个迷你的图形引擎，目前实现了部分光栅化的功能.
- 实现了矩阵向量库，线性插值，***z-buffer*** 深度检测， ***MSAA*** 抗锯齿等功能.

### 编译环境
- ***C++20***
- ***Windows or Linux***

### 第三方库
- ***GLFW***
